# Lesson 09 — Memory 向量记忆系统：Memory Search 语义搜索

## 核心概念

`memory_search` 通过**混合搜索**找到记忆中的相关内容，即使表述与原始内容不同也能匹配。底层结合向量相似度（语义）和 BM25 关键词搜索。

## 关键设计

### 搜索流程
```
Query → Embedding + Tokenize → Vector Search + BM25 Search → Weighted Merge → Results
```
- **向量搜索**：理解语义，"gateway host" 能匹配 "the machine running OpenClaw"
- **BM25 关键词搜索**：精确匹配，用于 ID、错误字符串、配置 key

### 搜索质量增强

**时间衰减（Temporal Decay）**
- 默认半衰期 30 天，旧笔记逐渐降低排名权重
- MEMORY.md 等常青文件不受衰减影响

**MMR（最大边际相关性）**
- 减少冗余结果，确保 top 结果覆盖不同主题

### 多模态记忆
使用 Gemini Embedding 2，可索引图像和音频，搜索查询仍为文本。
