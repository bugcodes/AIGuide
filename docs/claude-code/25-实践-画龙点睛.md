### 23.0 引言：为什么需要定制？

Claude Code 开箱即用已经很强大了，但默认配置只能发挥它 40% 的潜力。通过深度定制，你可以让它：

```
定制前的 Claude Code：
├─ 通用的任务处理
├─ 固定的工作流程
└─ 有限的上下文理解

定制后的 Claude Code：
├─ 深度理解你的代码库
├─ 自动化你的专属工作流
├─ 记住你的偏好和习惯
└─ 持续学习你的领域知识
```

这一章，我们将从配置文件、Skills 开发、插件系统、性能优化四个维度，深度解锁 Claude Code 的定制能力。

## AI 使用规范

### 何时使用 Claude Code
✅ **适合使用：**
- 代码审查和重构
- 测试用例生成
- Bug 定位和分析
- API 文档生成
- 性能优化建议
- 学习新技术的概念解释

❌ **不适合使用：**
- 直接修改生产环境代码（必须经过 Code Review）
- 生成涉及支付、权限的核心逻辑（必须人工审核）
- 一次性生成大量代码（应分步进行）
- 处理敏感数据（用户信息、密码等）

### Prompt 规范
```markdown
1. 提供足够的上下文（相关代码、错误信息、约束条件）
2. 明确输出格式要求
3. 说明优先级（哪些必须做到，哪些可以妥协）
4. 设定边界（什么情况不处理，什么情况要警告）
```

### 安全红线
```markdown
🚨 绝对禁止：
- 向任何 AI 服务发送用户密码或密钥
- 在 Prompt 中包含完整的用户个人信息
- 让 AI 绕过权限检查逻辑
- 生成可以绕开安全机制的代码

⚠️ 需要人工复核：
- 支付相关逻辑
- 权限检查逻辑
- 认证/授权逻辑
- 涉及数据修改的批量操作
```

#### 23.2.1 SKILL.md 格式详解

Skills 是 Claude Code 最强大的扩展机制。一个 Skill 本质上是一个带有元数据的 Prompt 包：

```markdown
# SKILL.md

## name
skill-identifier-name

## description
一段简短描述，说明这个 Skill 做什么（1-2 句话）

## triggers
- /command-name
- 当用户提到特定关键词时触发
- 在特定文件类型中自动激活

## parameters
- param_name: description, type, required
- output_format: 输出格式要求, string, no

## context
- 从哪些来源获取上下文信息
- 项目的 .claude/CLAUDE.md
- 相关文件自动注入

## handler
\`\`\`
处理逻辑
\`\`\`

## examples
### 示例 1
用户输入：
```
/git-commit-helper
```

Claude 输出：
```
[生成的 commit message]
```

### 示例 2
...
```

#### 23.2.2 Skill 示例：智能 Git Commit Helper

```markdown
# skills/git-commit-helper/SKILL.md

# SKILL.md

## name
git-commit-helper

## description
分析 Git diff 并生成符合 Conventional Commits 规范的提交信息。支持多语言项目，自动识别变更类型。

## triggers
- /git-commit-helper
- /commit
- "帮我写 commit message"
- "generate commit message"

## parameters
- scope: 变更范围（可选，如：auth, payment, api）
- type_override: 覆盖自动识别的变更类型（可选）
- message_length: 消息长度偏好（short/medium/long）

## context
- 自动执行 `git diff --cached` 获取暂存区变更
- 自动执行 `git diff HEAD~3` 获取最近 3 次提交风格
- 读取项目根目录的 COMMIT_CONVENTION.md（如果存在）

## handler
\`\`\`bash
# 获取暂存区的 diff
git diff --cached --stat

# 获取详细 diff
git diff --cached

# 检查是否有 commit convention 配置
if [ -f .commitlintrc.js ] || [ -f .czrc ]; then
    echo "PROJECT_HAS_COMMIT_CONVENTION=true"
fi
\`\`\`

## examples

### 示例 1：标准使用
用户输入：
```
/git-commit-helper
```

Claude 行为：
```
1. 执行 git diff --cached 获取变更
2. 分析变更内容，识别变更类型
3. 生成符合规范的 commit message
4. 显示预览，等待用户确认或修改
```

### 示例 2：指定 scope
用户输入：
```
/git-commit-helper --scope payment
```

Claude 输出：
```
feat(payment): add WeChat Pay integration

- Implement WeChat Pay SDK wrapper
- Add payment callback handler
- Update order status on successful payment
- Add unit tests for payment service
```

### 示例 3：复杂变更
用户输入：
```
/git-commit-helper
```

Git diff 内容：
```diff
+import stripe
+from stripe import PaymentIntent

 def process_payment(order_id: str, amount: int):
-    result = external_api.call("/pay", {"order_id": order_id, "amount": amount})
-    if result["code"] == 0:
-        OrderService.update_status(order_id, "PAID")
-    return result
+    try:
+        intent = stripe.PaymentIntent.create(
+            amount=amount,
+            currency='usd',
+            metadata={'order_id': order_id}
+        )
+        return {"status": "requires_action", "client_secret": intent.client_secret}
+    except stripe.error.CardError as e:
+        logger.error(f"Payment failed: {e}")
+        return {"status": "failed", "error": str(e)}
```

Claude 输出：
```
feat(payment): refactor payment processing to Stripe SDK

BREAKING CHANGE: Payment API response format changed
- Replace external API call with Stripe SDK integration
- Add PaymentIntent creation with order metadata
- Implement error handling for card errors
- Return client_secret for frontend 3DS authentication
```

## output_format
\`\`\`markdown
{type}({scope}): {subject}

{body}

{footer}
\`\`\`

## conventions
如果项目没有自定义规范，使用以下默认规则：

### Type 分类
- feat: 新功能
- fix: Bug 修复
- docs: 文档更新
- style: 代码格式（不影响功能）
- refactor: 重构（非新功能非修复）
- perf: 性能优化
- test: 测试相关
- chore: 构建/工具变更
- revert: 回滚

### Subject 规则
- 不超过 50 字符
- 使用动词原形开头
- 不使用句号结尾
- 使用中文描述

### Body 规则（可选）
- 解释 What 和 Why，不解释 How
- 每行不超过 72 字符

### Footer 规则（可选）
- Breaking Changes: 以 "BREAKING CHANGE:" 开头
- 关联 Issue: "Closes #123"
```

#### 23.2.3 Skill 示例：项目健康度检查

```markdown
# skills/project-health-checker/SKILL.md

# SKILL.md

## name
project-health-checker

## description
全面检查项目健康度，包括代码质量、安全漏洞、测试覆盖、性能问题等，生成可操作的改进报告。

## triggers
- /health-check
- /project-health
- "检查项目健康度"
- "项目质量报告"

## parameters
- depth: 检查深度（quick/normal/deep），默认 normal
- focus_areas: 重点检查领域（可多选：quality/security/performance/coverage）

## handler
\`\`\`bash
# 收集项目信息
echo "=== Project Health Check ==="
echo "Project: $(basename $(pwd))"
echo "Node: $(node --version 2>/dev/null || echo 'N/A')"
echo "Python: $(python3 --version 2>/dev/null || echo 'N/A')"

# 代码统计
echo -e "\n=== Code Stats ==="
find . -name "*.py" -o -name "*.ts" -o -name "*.js" | grep -v node_modules | wc -l

# Git 状态
echo -e "\n=== Git Status ==="
git status --short 2>/dev/null || echo "Not a git repo"

# 测试覆盖率
echo -e "\n=== Test Coverage ==="
if [ -f coverage/coverage.txt ]; then
    cat coverage/coverage.txt
elif [ -f coverage/lcov.info ]; then
    grep "Overall" coverage/lcov.info || echo "Run tests with --coverage first"
fi

# 安全扫描
echo -e "\n=== Security Scan ==="
if [ -f package-lock.json ]; then
    npm audit --dry-run 2>/dev/null | head -20
elif [ -f requirements.txt ]; then
    pip-audit --dry-run 2>/dev/null | head -20
fi

#### 23.3.2 上下文优化策略

```python
class ContextOptimizer:
    """
    上下文优化器
    目标：在有限的 Token 预算内最大化有效信息量
    """

    def __init__(self, max_tokens: int = 200000):
        self.max_tokens = max_tokens
        # Token 预算（保留 20% 给输出）
        self.input_budget = int(max_tokens * 0.8)

    def optimize_context(self, files: list[dict], task: str) -> list[dict]:
        """
        优化上下文文件列表

        策略：
        1. 按相关性排序
        2. 截断不重要的文件
        3. 优先保留核心文件
        """
        # 计算每个文件的相关性分数
        scored_files = []
        for f in files:
            relevance = self._calculate_relevance(f, task)
            size_penalty = self._calculate_size_penalty(f)
            score = relevance * (1 - size_penalty)
            scored_files.append((score, f))

        # 按分数排序
        scored_files.sort(key=lambda x: x[0], reverse=True)

        # 选择能放入预算的文件
        selected = []
        current_tokens = 0

        for score, f in scored_files:
            file_tokens = self._estimate_tokens(f['content'])
            if current_tokens + file_tokens <= self.input_budget:
                selected.append(f)
                current_tokens += file_tokens
            else:
                # 尝试截断而不是直接丢弃
                truncated = self._truncate_file(f, self.input_budget - current_tokens)
                if truncated:
                    selected.append(truncated)

        return selected

    def _calculate_relevance(self, file: dict, task: str) -> float:
        """计算文件与任务的相关性"""
        # 关键词匹配
        keywords = self._extract_keywords(task)
        content_keywords = self._extract_keywords(file['content'])

        matches = sum(1 for k in keywords if k in content_keywords)
        return matches / len(keywords) if keywords else 0.5

    def _truncate_file(self, file: dict, remaining_tokens: int) -> dict:
        """截断文件以适应预算"""
        max_lines = int(remaining_tokens / 3)  # 假设每行 3 tokens

        if file['line_count'] <= max_lines:
            return file

        return {
            **file,
            'content': '\n'.join(file['content'].split('\n')[:max_lines]),
            'truncated': True,
            'original_lines': file['line_count'],
            'kept_lines': max_lines
        }
```

#### 23.3.3 长期记忆系统

```python
import json
import hashlib
from datetime import datetime, timedelta
from pathlib import Path

class LongTermMemory:
    """
    长期记忆系统
    在会话之间持久化关键信息
    """

    def __init__(self, memory_dir: str = "~/.claude/memory"):
        self.memory_dir = Path(memory_dir).expanduser()
        self.memory_dir.mkdir(parents=True, exist_ok=True)

        # 索引文件
        self.index_file = self.memory_dir / "index.json"
        self.index = self._load_index()

    def _load_index(self) -> dict:
        """加载索引"""
        if self.index_file.exists():
            return json.loads(self.index_file.read_text())
        return {"entries": [], "last_cleanup": None}

    def _save_index(self):
        """保存索引"""
        self.index_file.write_text(json.dumps(self.index, indent=2))

    def store(
        self,
        key: str,
        value: str,
        tags: list[str] = None,
        ttl_days: int = 30
    ) -> str:
        """
        存储记忆

        Args:
            key: 记忆的标识键
            value: 记忆内容
            tags: 标签列表，用于检索
            ttl_days: 过期天数

        Returns:
            记忆 ID
        """
        memory_id = hashlib.sha256(f"{key}{datetime.now()}".encode()).hexdigest()[:12]

        entry = {
            "id": memory_id,
            "key": key,
            "content": value,
            "tags": tags or [],
            "created_at": datetime.now().isoformat(),
            "expires_at": (datetime.now() + timedelta(days=ttl_days)).isoformat(),
            "access_count": 0,
            "last_accessed": None
        }

        # 保存记忆文件
        memory_file = self.memory_dir / f"{memory_id}.json"
        memory_file.write_text(json.dumps(entry, indent=2))

        # 更新索引
        self.index["entries"].append({
            "id": memory_id,
            "key": key,
            "tags": tags
        })
        self._save_index()

        return memory_id

    def recall(self, key: str = None, tag: str = None, limit: int = 10) -> list[dict]:
        """
        召回记忆

        Args:
            key: 精确键匹配
            tag: 标签匹配
            limit: 返回数量限制

        Returns:
            匹配的记忆列表
        """
        results = []

        for entry_meta in self.index["entries"]:
            if key and entry_meta["key"] != key:
                continue
            if tag and tag not in entry_meta.get("tags", []):
                continue

            # 读取完整记忆
            memory_file = self.memory_dir / f"{entry_meta['id']}.json"
            if memory_file.exists():
                entry = json.loads(memory_file.read_text())

                # 检查过期
                if datetime.fromisoformat(entry["expires_at"]) > datetime.now():
                    # 更新访问统计
                    entry["access_count"] += 1
                    entry["last_accessed"] = datetime.now().isoformat()
                    memory_file.write_text(json.dumps(entry, indent=2))

                    results.append(entry)

        # 按访问频率和最近访问时间排序
        results.sort(
            key=lambda x: (x["access_count"], x["last_accessed"] or ""),
            reverse=True
        )

        return results[:limit]

    def search(self, query: str, limit: int = 10) -> list[dict]:
        """
        语义搜索记忆
        （简化版，实际可结合 Embeddings 实现更智能的搜索）
        """
        query_keywords = set(query.lower().split())
        results = []

        for entry_meta in self.index["entries"]:
            memory_file = self.memory_dir / f"{entry_meta['id']}.json"
            if not memory_file.exists():
                continue

            entry = json.loads(memory_file.read_text())

            # 简单关键词匹配
            content_words = set(entry["content"].lower().split())
            tag_words = set(" ".join(entry.get("tags", [])).lower().split())

            score = len(query_keywords & content_words) + \
                    len(query_keywords & tag_words) * 2

            if score > 0:
                results.append((score, entry))

        results.sort(key=lambda x: x[0], reverse=True)
        return [r[1] for r in results[:limit]]

    def forget(self, memory_id: str) -> bool:
        """删除记忆"""
        memory_file = self.memory_dir / f"{memory_id}.json"

        if memory_file.exists():
            memory_file.unlink()

            # 从索引中移除
            self.index["entries"] = [
                e for e in self.index["entries"]
                if e["id"] != memory_id
            ]
            self._save_index()
            return True

        return False

    def cleanup_expired(self) -> int:
        """清理过期记忆"""
        count = 0
        now = datetime.now()

        for entry_meta in self.index["entries"][:]:  # 复制列表避免修改迭代
            memory_file = self.memory_dir / f"{entry_meta['id']}.json"
            if memory_file.exists():
                entry = json.loads(memory_file.read_text())
                if datetime.fromisoformat(entry["expires_at"]) < now:
                    memory_file.unlink()
                    self.index["entries"].remove(entry_meta)
                    count += 1

        self.index["last_cleanup"] = now.isoformat()
        self._save_index()
        return count
```

```python
import hashlib
import json
from datetime import datetime, timedelta
from pathlib import Path
import shelve

class PromptCache:
    """
    Prompt 缓存
    避免重复调用相同的 Prompt
    """

    def __init__(self, cache_dir: str = "~/.claude/cache"):
        self.cache_dir = Path(cache_dir).expanduser()
        self.cache_dir.mkdir(parents=True, exist_ok=True)

        # 使用 shelve 存储缓存（持久化字典）
        self.cache_file = self.cache_dir / "prompt_cache"
        self._cache = shelve.open(str(self.cache_file))

        # TTL 设置（默认 7 天）
        self.default_ttl = timedelta(days=7)

    def _make_key(self, prompt: str, context: dict = None) -> str:
        """生成缓存键"""
        content = json.dumps({
            "prompt": prompt,
            "context": context or {}
        }, sort_keys=True)

        return hashlib.sha256(content.encode()).hexdigest()

    def get(self, prompt: str, context: dict = None) -> Optional[dict]:
        """获取缓存"""
        key = self._make_key(prompt, context)

        if key in self._cache:
            entry = self._cache[key]

            # 检查是否过期
            expires_at = datetime.fromisoformat(entry["expires_at"])
            if expires_at > datetime.now():
                entry["hit_count"] = entry.get("hit_count", 0) + 1
                entry["last_accessed"] = datetime.now().isoformat()
                self._cache[key] = entry
                return entry["result"]
            else:
                # 删除过期条目
                del self._cache[key]

        return None

    def set(
        self,
        prompt: str,
        result: dict,
        context: dict = None,
        ttl: timedelta = None
    ) -> str:
        """设置缓存"""
        key = self._make_key(prompt, context)
        ttl = ttl or self.default_ttl

        entry = {
            "result": result,
            "created_at": datetime.now().isoformat(),
            "expires_at": (datetime.now() + ttl).isoformat(),
            "hit_count": 0,
            "last_accessed": None
        }

        self._cache[key] = entry
        return key

    def invalidate(self, pattern: str = None):
        """清除缓存"""
        if pattern:
            # 清除匹配模式的缓存
            keys_to_delete = [
                k for k in self._cache.keys()
                if pattern in self._cache[k].get("prompt", "")
            ]
            for k in keys_to_delete:
                del self._cache[k]
        else:
            # 清除全部
            self._cache.clear()

    def get_stats(self) -> dict:
        """获取缓存统计"""
        total = len(self._cache)
        expired = 0

        for entry in self._cache.values():
            if datetime.fromisoformat(entry["expires_at"]) < datetime.now():
                expired += 1

        total_hits = sum(e.get("hit_count", 0) for e in self._cache.values())

        return {
            "total_entries": total,
            "active_entries": total - expired,
            "expired_entries": expired,
            "total_hits": total_hits,
            "hit_rate": total_hits / total if total > 0 else 0
        }
```

class TokenManager:
    """
    Token 管理
    生成和验证 Claude Code 的会话 Token
    """

    def __init__(self, secret_key: str):
        self.secret_key = secret_key
        self.issuer = "claude-code-enterprise"

    def create_token(self, user: User, expires_hours: int = 24) -> str:
        """
        创建会话 Token

        Token Payload:
        - sub: user.id
        - email: user.email
        - teams: user.teams
        - roles: user.roles
        - quota: user.api_quota
        - iat: issued_at
        - exp: expires_at
        """
        now = datetime.utcnow()

        payload = {
            "sub": user.id,
            "email": user.email,
            "name": user.name,
            "teams": user.teams,
            "roles": user.roles,
            "quota": {
                "monthly": user.api_quota,
                "reset_at": (now.replace(day=1) + timedelta(days=32)).replace(day=1).isoformat()
            },
            "iat": now,
            "exp": now + timedelta(hours=expires_hours),
            "iss": self.issuer
        }

        return jwt.encode(payload, self.secret_key, algorithm="HS256")

    def verify_token(self, token: str) -> Optional[dict]:
        """验证 Token 并返回 Payload"""
        try:
            payload = jwt.decode(
                token,
                self.secret_key,
                algorithms=["HS256"],
                issuer=self.issuer
            )
            return payload
        except jwt.ExpiredSignatureError:
            # Token 过期
            return None
        except jwt.InvalidTokenError:
            # Token 无效
            return None

    def refresh_token(self, token: str) -> Optional[str]:
        """刷新 Token"""
        payload = self.verify_token(token)
        if not payload:
            return None

        # 创建新 Token
        user = User(
            id=payload["sub"],
            email=payload["email"],
            name=payload["name"],
            teams=payload["teams"],
            roles=payload["roles"],
            api_quota=payload["quota"]["monthly"]
        )

        return self.create_token(user)

#### 23.5.3 成本分摊与配额管理

```python
# enterprise/cost_allocation.py

from dataclasses import dataclass
from datetime import datetime
from typing import Dict
import sqlite3
from pathlib import Path

@dataclass
class UsageRecord:
    """使用记录"""
    user_id: str
    team_id: str
    project_id: str
    tokens_used: int
    cost_usd: float
    timestamp: datetime
    model: str
    task_type: str  # "code_generation", "review", "refactor", etc.

@dataclass
class QuotaStatus:
    """配额状态"""
    user_id: str
    team_id: str
    monthly_quota: int
    monthly_used: int
    remaining: int
    reset_at: datetime

class CostAllocator:
    """
    成本分摊系统
    按团队/项目/个人三个维度统计和分摊成本
    """

    def __init__(self, db_path: str = "~/.claude/usage.db"):
        self.db_path = Path(db_path).expanduser()
        self._init_db()

    def _init_db(self):
        """初始化数据库"""
        self.db_path.parent.mkdir(parents=True, exist_ok=True)

        conn = sqlite3.connect(self.db_path)
        cursor = conn.cursor()

        cursor.execute("""
            CREATE TABLE IF NOT EXISTS usage_records (
                id INTEGER PRIMARY KEY AUTOINCREMENT,
                user_id TEXT NOT NULL,
                team_id TEXT NOT NULL,
                project_id TEXT,
                tokens_used INTEGER NOT NULL,
                cost_usd REAL NOT NULL,
                timestamp TEXT NOT NULL,
                model TEXT NOT NULL,
                task_type TEXT NOT NULL
            )
        """)

        cursor.execute("""
            CREATE INDEX IF NOT EXISTS idx_usage_user_time
            ON usage_records(user_id, timestamp)
        """)

        cursor.execute("""
            CREATE INDEX IF NOT EXISTS idx_usage_team_time
            ON usage_records(team_id, timestamp)
        """)

        conn.commit()
        conn.close()

    def record_usage(self, record: UsageRecord):
        """记录使用量"""
        conn = sqlite3.connect(self.db_path)
        cursor = conn.cursor()

        cursor.execute("""
            INSERT INTO usage_records
            (user_id, team_id, project_id, tokens_used, cost_usd, timestamp, model, task_type)
            VALUES (?, ?, ?, ?, ?, ?, ?, ?)
        """, (
            record.user_id,
            record.team_id,
            record.project_id,
            record.tokens_used,
            record.cost_usd,
            record.timestamp.isoformat(),
            record.model,
            record.task_type
        ))

        conn.commit()
        conn.close()

    def get_user_quota_status(self, user_id: str) -> QuotaStatus:
        """获取用户配额状态"""
        conn = sqlite3.connect(self.db_path)
        cursor = conn.cursor()

        # 获取本月使用量
        now = datetime.now()
        month_start = now.replace(day=1).strftime("%Y-%m-%d")

        cursor.execute("""
            SELECT COALESCE(SUM(tokens_used), 0)
            FROM usage_records
            WHERE user_id = ? AND timestamp >= ?
        """, (user_id, month_start))

        monthly_used = cursor.fetchone()[0]
        conn.close()

        # 这里应该从配置中获取用户配额
        monthly_quota = 10_000_000  # 默认 10M tokens

        return QuotaStatus(
            user_id=user_id,
            team_id="",  # 从其他来源获取
            monthly_quota=monthly_quota,
            monthly_used=monthly_used,
            remaining=monthly_quota - monthly_used,
            reset_at=now.replace(day=1) + timedelta(days=32)
        )

    def get_team_cost_report(
        self,
        team_id: str,
        start_date: datetime,
        end_date: datetime
    ) -> dict:
        """生成团队成本报告"""
        conn = sqlite3.connect(self.db_path)
        cursor = conn.cursor()

        # 总体成本
        cursor.execute("""
            SELECT
                COUNT(*) as request_count,
                SUM(tokens_used) as total_tokens,
                SUM(cost_usd) as total_cost,
                model,
                task_type
            FROM usage_records
            WHERE team_id = ? AND timestamp BETWEEN ? AND ?
            GROUP BY model, task_type
        """, (team_id, start_date.isoformat(), end_date.isoformat()))

        rows = cursor.fetchall()
        conn.close()

        report = {
            "team_id": team_id,
            "period": {
                "start": start_date.isoformat(),
                "end": end_date.isoformat()
            },
            "by_model": {},
            "by_task_type": {},
            "total": {
                "requests": 0,
                "tokens": 0,
                "cost_usd": 0.0
            }
        }

        for row in rows:
            request_count, tokens, cost, model, task_type = row

            report["by_model"][model] = {
                "requests": request_count,
                "tokens": tokens,
                "cost_usd": cost
            }

            report["by_task_type"][task_type] = {
                "requests": request_count,
                "tokens": tokens,
                "cost_usd": cost
            }

            report["total"]["requests"] += request_count
            report["total"]["tokens"] += tokens
            report["total"]["cost_usd"] += cost

        return report
```
