## Claude Code 四大核心能力

1. **对话（Dialogue）** — 自然语言交互
2. **工具（Tools）** — Read/Write/Bash/Glob/Grep/WebFetch
3. **上下文（Context）** — 动态上下文窗口
4. **记忆（Memory）** — 三层记忆系统

---

## 基础 vs 高级：分水岭

**基础用法（当搜索引擎用）：**
```
> 帮我写一个 Python 快速排序
-> 给你代码，结束
```

**高级用法（任务委托）：**
```
> 我的项目是 FastAPI 后端
> 帮我分析 src/ 目录下的安全漏洞
-> 读取多个文件 -> 分析 -> 跨文件关联 -> 生成报告
```

**关键区别：基础用法 = 问答，高级用法 = 任务委托**
