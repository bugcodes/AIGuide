## 课程完整内容

### 24.0 引言：站在浪潮之巅

2021 年，GitHub 推出了 Copilot，代码补全第一次变得"智能"。

2023 年，Claude Code 诞生，AI 不再只是"补全代码"，而是能够"理解任务"。

2024 年，多 Agent 协作、自主规划、工具调用——AI 编码助手开始具备真正的"系统性思维"。

2025 年，我们正站在一个转折点上：AI 不是要取代开发者，而是要**重新定义开发者的角色**。

这一课，我们不聊具体的技术操作，而是聊一些更宏观的东西：**我们正在经历什么，未来会走向哪里，以及我们该如何应对**。
